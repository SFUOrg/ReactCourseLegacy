# 🚀 ReactCourseLegacy — учебный сайт по основам React

**ReactCourseLegacy** — это лёгкий, визуально приятный учебный сайт, созданный для тех, кто хочет быстро разобраться в ключевых концепциях React, не устанавливая ни npm, ни сборщики, ни сам React.
Проект полностью статический и состоит из HTML-страниц, изображений и чистого CSS/JS, оформленных как мини-учебник.

---

## 🎯 Что это за проект?

Это не просто набор страниц — это **мини-курс**, каждая тема которого вынесена на отдельную страницу:

- компоненты и JSX

- props и state

- обработка событий

- условный рендеринг

- списки и ключи

- формы и контролируемые компоненты

- жизненный цикл и хуки

И всё это подано в современном визуальном стиле — аккуратные картинки, мягкие цвета, понятная структура.

## 🌟 Особенности

- **Полностью статический проект** — открываешь index.html и готово.

- **Интуитивная навигация** между уроками.

- **Каждая тема визуализирована** собственным изображением.

- **Современный UI** — нейтральная палитра, чистый макет, понятная иерархия текста.

- **Идеально для студентов** или для собственных пет-проектов / портфолио.

## 🚀 Как запустить проект

Проект не требует зависимостей. Чтобы запустить:

```bash
git clone https://github.com/<username>/<repo-name>.git
```

Затем открыть в браузере:

```
index.html
```
