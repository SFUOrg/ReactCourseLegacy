Lifecycle Methods and Hooks

Understanding Component Lifecycle and Hooks

In class components, lifecycle methods are special methods that get called at specific points in a component's lifecycle, such as when it mounts, updates, or unmounts. The main lifecycle methods are:

  • componentDidMount - runs after the component is rendered to the DOM
  • componentDidUpdate - runs after the component updates
  • componentWillUnmount - runs right before the component is removed from the DOM

Hooks allow you to use state and other React features in functional components. The most common hooks are:

  • useState - for managing state
  • useEffect - for performing side effects (replaces lifecycle methods)
  • useContext - for accessing context values

The useEffect hook can handle the same scenarios as lifecycle methods. You can run effects after render by default, or conditionally run them when specific values change.

Example: Using useEffect Hook

import { useState, useEffect } from 'react';
function Example() {
    const [count, setCount] = useState(0);

    // Similar to componentDidMount and componentDidUpdate
    useEffect(() => {
      // Update the document title using the browser API
      document.title = `You clicked ${count} times`;
    });

    return (
      <div>
        <p>You clicked {count} times</p>
        <button onClick={() => setCount(count + 1)}>
          Click me
        </button>
      </div>
    );
}

Example: useEffect with Dependency Array

import { useState, useEffect } from 'react';

function FriendStatus(props) {
  const [isOnline, setIsOnline] = useState(null);

  useEffect(() => {
    function handleStatusChange(status) {
      setIsOnline(status.isOnline);
    }

    // Similar to componentDidMount
    ChatAPI.subscribeToFriendStatus(props.friend.id, handleStatusChange);

    // Similar to componentWillUnmount
    return () => {
      ChatAPI.unsubscribeFromFriendStatus(props.friend.id, handleStatusChange);
    };
  }, [props.friend.id]); // Only re-run if props.friend.id changes

  if (isOnline === null) {
    return 'Loading...';
  }
  return isOnline ? 'Online' : 'Offline';
}

Example: Multiple useEffect Hooks

function FriendStatusWithCounter(props) {
  const [count, setCount] = useState(0);
  const [isOnline, setIsOnline] = useState(null);

  useEffect(() => {
    document.title = `You clicked ${count} times`;
  }); // No dependency array - runs after every render

  useEffect(() => {
    function handleStatusChange(status) {
      setIsOnline(status.isOnline);
    }

    ChatAPI.subscribeToFriendStatus(props.friend.id, handleStatusChange);
    return () => {
      ChatAPI.unsubscribeFromFriendStatus(props.friend.id, handleStatusChange);
    };
  }, [props.friend.id]); // Only runs when props.friend.id changes

  // ... component JSX
}
