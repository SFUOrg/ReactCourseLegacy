React Fundamentals / Conditional Rendering

Understanding Conditional Rendering

Conditional rendering in React works the same way conditions work in JavaScript. You can use JavaScript operators like if or the ternary operator to create elements representing the current state, and let React update the UI to match them.

There are several ways to conditionally render content in React:

  • Using if statements in the render method
  • Using inline conditional with the ternary operator
  • Element variables
  • Inline if with logical && operator

React evaluates conditions and only renders the elements that should be displayed based on the current state.

Example: Conditional Rendering with if-else

function Greeting(props) {
  const isLoggedIn = props.isLoggedIn;
  
  if (isLoggedIn) {
    return <h1>Welcome back!</h1>;
  }
  return <h1>Please sign up.</h1>;
}

Example: Conditional Rendering with Ternary Operator

function Greeting(props) {
  const isLoggedIn = props.isLoggedIn;
  
  return (
    <div>
      {isLoggedIn ? (
        <h1>Welcome back!</h1>
      ) : (
        <h1>Please sign up.</h1>
      )}
    </div>
  );
}

Example: Conditional Rendering with Logical &&

function Mailbox(props) {
  const unreadMessages = props.unreadMessages;
  
  return (
    <div>
      <h1>Hello!</h1>
      {unreadMessages.length > 0 && 
        <h2>
          You have {unreadMessages.length} unread messages.
        </h2>
      }
    </div>
  );
}
