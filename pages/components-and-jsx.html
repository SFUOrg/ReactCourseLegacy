React Fundamentals / Components and JSX

Understanding Components

React components are independent and reusable bits of code that describe a part of the user interface. Components can be thought of as JavaScript functions that receive inputs (called props) and return React elements describing what should appear on the screen.

Components can be either functional (stateless) or class-based (stateful). Functional components are simpler and are preferred for components that only need to render UI based on props.

JSX Syntax

JSX stands for JavaScript XML and allows us to write HTML-like code in our JavaScript files. It is a syntax extension for JavaScript that looks similar to HTML but is transformed into regular JavaScript during the build process.

JSX allows embedding expressions in curly braces, making it easy to combine JavaScript logic with HTML structure.

Example

Here's a simple example of a React component using JSX:

function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

// Using the component
const element = <Welcome name="Sarah" />;

// Renders: <h1>Hello, Sarah!</h1>
